- [x] 建立 mock 仲裁员列表，至少 3 名有效仲裁员。

- [x] 在实现动画前检查当前 Codex 环境是否提供 GSAP skill；若提供，必须先加载并遵守该 skill。
- [x] 前端依赖中加入 `gsap`，仲裁中心页面动画通过 GSAP 实现。
